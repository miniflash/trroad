# trroad

Official implementation of "Passable Area Segmentation for Open-Pit Mine Road from Vehicle Perspective":

This project is written based on [mmsegmentation](https://github.com/open-mmlab/mmsegmentation.git).

The mine road scapes segmentation (MRSS) dataset is partially available in this project, complete dataset will coming
soon.
